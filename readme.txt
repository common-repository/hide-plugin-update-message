== Installation ==
1. Upload the folder "hide-pluging-update-message" to "/wp-content/plugins/" path.
2. Activate the plugin through the "Plugins" menu in WordPress backend
3. After activating the plugin, upgrade message will be hide.
4. Enjoy!
